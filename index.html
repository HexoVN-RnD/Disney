R&D PROJECT OVERVIEW
Interactive Games & Technology

Core technology analysis report from the Disney "Wonder of Friendship" project and R&D directions (social media reverse-engineering) for future shows.

Part 1: Disney Project

Breaking down the hardware & software technologies behind the core interactive spaces of "Wonder of Friendship".

Lilo & Stitch:
Secrets in the Sand

Core: Warm-up experience. Players interact with a virtual sand screen, drawing or making footprints to discover Stitch's surprise appearance.

  • Surface Scanning: Using 2D Lidar (Hokuyo) placed at the floor edge to create a horizontal "laser curtain", detecting precise touch coordinates on the virtual sand.
  • Signal Processing: Point Cloud data is sent via TUIO/OSC protocol to the Media Server.
  • Physics Simulation: TouchDesigner or Unreal Engine (Niagara) calculates sand particles being swept away in real-time and maps the image onto the floor.

Toy Story:
Let's Save Woody

Core: Using body movement back and forth to "erase" the giant Etch-A-Sketch screen layer.

  • Camera Tracking: Using overhead Depth Cameras (Azure Kinect, Intel RealSense).
  • Optical Flow: Blob data fed into TouchDesigner creates a virtual brush to erase the mask layer.
  • Optimization: Background subtraction to identify only humans.

The Lion King:
Music in the Jungle

The Jungle zone requires players to hit tree-trunk drums in sync to trigger lights and visuals.

  • Hardware: Piezo sensors attached inside the drums create analog signals.
  • Microcontroller: Arduino reads signals, converts to MIDI standard.
  • A/V Sync: MIDI sent to Resolume/TouchDesigner to trigger video and LED Pixels.

Frozen:
Reuniting with Olaf

Core Experience: Players throw physical snowballs at the interactive ice wall, shattering it gradually to reveal Olaf.

  • Collision Detection: Scanning the wall surface with 2D Lidar (Hokuyo) creates an invisible "laser curtain".
  • Coordinate Calculation: When a snowball breaks the laser, Lidar returns exact (X, Y) touch coordinates.
  • Particle Physics: Coordinates sent via UDP to Unreal Engine. Spawns particle systems of shattered ice with realistic physics.

Exploration & Rhythm

Alice in Wonderland

Gameplay: Find hidden spots, use the "Friendship Medallion" to activate the Cheshire Cat.

Tech R&D: Using RFID / NFC technology hidden behind decor walls. Readers send API commands via LAN to trigger Mapping video projection.

Mickey & Friends

Gameplay: Dance Challenge, players dance following character silhouettes to score points.

Tech R&D: Applying AI Pose Estimation (MediaPipe / Azure Kinect). Comparing bone joint angle vectors with sample animations for real-time scoring.

Transition Zone:
SketchBot Station

Core Experience: A fun rest stop where guests take photos and receive a caricature portrait drawn live by a robotic arm.

  • AI Face Processing: Software captures and analyzes faces, converting them to vector line art.
  • Robotic Kinematics: G-code data controls a miniature industrial robotic arm drawing on paper in real-time.
  • Souvenir: Drawn on pre-printed Disney frame paper, a unique personalized gift.

Part 2: Control Systems

Comprehensive management app ecosystem: From payment, zone control to central technical processing (Core Show Control).

1. Currency Management:
App Payment & Medallion

Core: Payment and top-up system (Coin/Credit) for the interactive necklace (Friendship Medallion).

  • Multi-channel Coin Top-up: Integrating payment APIs (Momo, VNPay, Visa) at Kiosks/Ticketing booths to buy Credit.
  • Write Data RFID: App to scan and directly write balance and customer ID onto the built-in NFC/RFID chip in the Medallion.
  • Cloud Sync: Synchronize transaction history (e.g., Gacha plays, buying drinks at The Hunny Pot Cafe) to the Cloud system (Firebase/AWS) in real-time.

2. Zone Operations:
App Zone Manager

Role: Intuitive control interface (UI/UX) on Tablet/PC for operators at each zone.

  • Power Management: All-in-one button to turn on/off all projectors, screens, and station PCs via LAN (Wake-on-LAN, PJLink).
  • Scene Transition: Control game flow smoothly. Example: Transition from 'Waiting Scene' -> 'Start Game' -> 'End & Receive Gem'.
  • Health Dashboard: Monitor device operational status, track sensors, and alert on network disconnection.

3. Heart of the System:
App Show Control

Role: Background application running on the Server, processing deep technical logic and syncing all hardware signals.

  • Mapping & Sensor Calib: Contains tools for sensor calibration (Lidar, Depth Camera) and Projection Mapping adjustment (Warping/Blending).
  • Lighting Trigger: Outputs standard DMX / Art-Net signals to control LED lights, syncing intensity precisely with the Game rhythm.
  • Mechanical Systems (SFX): Sends triggers (Relay, TCP/OSC) to auto open/close curtains, lifting motors, smoke/wind machines for practical effects.
